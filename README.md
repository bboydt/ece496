# ECE 4960
Clemson - Integrated Systems Design 2

## Building

To build and use the project the following program suites are needed:

- `scons` it's the build system
- `oss-cad-suite` includes `yosys` and other programs for gateware synthesis and bitstream creation.
- GNU bin tools compiled for `riscv64-unknown-elf`

Fetch the git submodules and then run `scons` from the root directory. This will build the project in `build`.

The commit `rt` refrences does not work as of 5/1/2024. This is because a local fork was used to port it to `risc-v`. If attempting to compile in the future check the `rt` repo for risc-v support and update the commit in the submodule.
